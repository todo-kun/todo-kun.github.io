# Contributing

## Development flow

1. Create a branch from `main`
2. Make the change
3. Run:
   - `npm run typecheck`
   - `npm run lint`
   - `npm run test`
   - `npm run build`
4. Open a pull request

## Setup

1. Install dependencies with `npm install`
2. Start the app with `npm run dev`
3. Configure Google and app settings from the browser settings panel

## Pull requests

- Keep changes focused
- Describe user-facing impact
- Include mobile-width checks for UI changes
- Mention whether Google-connected behavior was verified
